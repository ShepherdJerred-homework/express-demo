Create a complete Express application.

  1. It should be a package following the conventions we established in the previous homeworks. Don't forget:
    • List all dependencies in package.json
    • Your test command should be mocha
    • Create a test directory and put your test cases inside it
    • Test cases are recommended but not required
  2. Your web application should listen to port 8000. When run, it should output the message, "Listening on port 8000...".
  3. Write a handler for the URL /hello
    • Create a plain-text response that says, “Hello, Name”
    • The name should come from the name parameter in the query string. Use the req.query object to find the value of this property
    • If no name parameter is given then use “Somebody” as the name
    • Don't forget to set the status to 200
  4. Write a handler for the URL /picture
    • Create a directory in your project named static
    • Store an image file in this directory
    • The handler should use res.sendFile to send the image file as the response to the request.
    • Don't forget to set the status to 200
  5. Write a default handler
    • Should be registered last: to be called if no other handler has handled the request
    • Should have a 404 status code
    • Respond with a web page that says the following: 
      The requested URL /whatever was not found on this server.
      Replace /whatever with the actual URL requested
    • You will need to construct the HTML as a giant string in your handler so that you can customize it with the URL.
    • Remember to set the content type to text/html
